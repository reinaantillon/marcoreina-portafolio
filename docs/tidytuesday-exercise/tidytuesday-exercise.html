From this information we can see the most “common” coordinates for sightings for DIO is 150, -40 with 7749 birds observed there.

I am going to use this to measure “spread” I will use this as my base coordinate group and measure the distance between this high frequency coordinate to the actual

ANALYSIS:

Hypothesis: The spread of DIO is greater in warmer months than in cold months/seasons.

Question: Is the spread getting smaller over the years?

#Let's calculate distance

#Setting the main coordinate bins
lat_i <- -40
lon_i <- 150

# As the Earth is a globe, we need a special formula to calculate distance from A to B
library(geosphere)

trends_df <- trends_df %>%
  mutate(
    spreadkm = 1 + distHaversine(
      cbind(lon_i, lat_i), 
      cbind(lon_bin, lat_bin)
    )/1000,
    spread_rounded = round(spreadkm/100, digits = 0) * 100
  ) 

#Histograms to understand spread across both waves
trends_df %>%
  ggplot(aes(x = spread_rounded)) + 
  geom_histogram(binwidth = 1000) +
  facet_wrap(~ wave)
#Seasons in wave 1
trends_df %>%
  filter(wave == 1) %>%
  ggplot(aes(x = spread_rounded)) + 
  geom_histogram(binwidth = 1000) +
  facet_wrap(~ season)
#Seasons in wave 2
trends_df %>%
  filter(wave == 2) %>%
  ggplot(aes(x = spread_rounded)) + 
  geom_histogram(binwidth = 1000) +
  facet_wrap(~ season)

From this information we can see that wave 1 have a wider spread in the autumn and summer. Wave 2 seems more compact across seasons, spring has the wider spread. Which it confirms what we observed on the maps.

All of the trends are right skewed regardless of the season or wave.

trends_df$spread_log2 <- log(trends_df$spreadkm, base = exp(2)) + 1

#Creating a null linear model
lm_null <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ 1, data = trends_df)

#Including all variables and a stepwise approach (reverse)
lm_fit1 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ wave + season + year + month + count + lat_bin + lon_bin, data = trends_df)

lm_fit2 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ wave + season + year + month + lat_bin + lon_bin, data = trends_df)

lm_fit3 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ wave + season + year + lat_bin + lon_bin, data = trends_df)

lm_fit4 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ wave + season + lat_bin + lon_bin, data = trends_df)

lm_fit5 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ season, data = trends_df)

lm_fit6 <- linear_reg() %>%
  set_engine("lm") %>% 
  fit(spread_log2 ~ wave, data = trends_df)

Let’s get the predictions

#Predidiction and differences for each fit
trends_df <- trends_df %>%
  mutate(
    pred0 = predict(lm_null, new_data = trends_df, type = "numeric")$.pred,
    pred1 = predict(lm_fit1, new_data = trends_df, type = "numeric")$.pred,
